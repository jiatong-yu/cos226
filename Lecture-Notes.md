## Soring
**Selection Sort** \
independent of given array order.  
~ N^2 / 2 compares  
~ N exch  
Stable? No (*swapping kth element with smallest*)  
In-Place? Yes  

**Insertion Sort** \
dependent of given array order  
~ Compares: *avg* N^2 / 4; *worst* N^2 / 2  
~ Exch: *avg* N^2 / 4; *worst* N^2 / 2  
Stable? Yes  
In-Place? Yes
